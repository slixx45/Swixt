# Swixt Browser

Dein Browser. Neu definiert. Ein moderner, datenschutzfreundlicher Browser mit Tracker-Blocking und hochentwickeltem Premium-Design.

---

## Features

* Tracker-Blocking: Über 50 bekannte Tracking-Domains (wie Google Analytics, Facebook Pixel etc.) werden automatisch im Hintergrund blockiert.
* Vollständig anpassbar: Ändere Akzentfarben, Hintergrundbilder, Transparenz und die Stärke des Glassmorphismus direkt in den Einstellungen.
* Schnellthemen: Wechsel mit nur einem Klick zwischen Themes wie Standard, Midnight, Rose, Ocean, Forest und Glassmorphic.
* Lesezeichen-System: Schnelles Hinzufügen mit Strg + D, Rechtsklick zum Entfernen oder Öffnen in neuen Tabs.
* Multi-Tab Browsing: Sanfte Tab-Animationen. Schnelles Öffnen mit Strg + T und Schließen mit Strg + W.
* Glassmorphismus-UI: Modernste Benutzeroberfläche mit edlen Unschärfe- und Transparenzeffekten.

2. Starte die Installation per Doppelklick.

1. Lade die aktuelle app.AppImage herunter.
